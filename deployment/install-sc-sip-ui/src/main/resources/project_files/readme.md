# Project sc-sip-ui : project_files/ readme #

This folder stores every file that we need to install or update the project our of server themselves.
Content for this folder will be copied ans subsituted in ${PROJECT_HOME} winth the installation or the update process.

This will happens at the end of the installation
- after product and binaries are installed
- before applciation specific script are executed
- before server and service are started
